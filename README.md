# 🤖 InnovaFP-IA

Aquest projecte és una base pràctica per aprendre a integrar Intel·ligència Artificial en aplicacions web (Frontend), desenvolupat en el marc del programa **InnovaFP**. 

Aquí trobareu l'estructura inicial amb `index.html` i `index.css` per començar a connectar els vostres dissenys amb models de llenguatge (IA).

---

### 1. Clonar el repositori
